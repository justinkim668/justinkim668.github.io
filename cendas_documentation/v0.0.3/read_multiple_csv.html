cendas.read_multiple_csv

cendas.read_multiple_csv(list: List)

Parameters/Arguments

  • list
    • A list of file paths indicating where .csv files are located

Description

Takes a list of file paths for .csv files as an argument. Returns a list of pandas dataframes with the data from the .csv files.

Why This Method is Useful

Currently, the pandas.read_csv method is only able to hande one .csv file at a time. The cendas.read_multiple_csv method allows a user to read mutlple .csv files into pandas dataframes at once.
