cendas.state_column

cendas.state_column(data_frame: pandas.DataFrame(), tractcode_column: str)

Parameters/Arguments

  • data_frame
    • A pandas dataframe.
  • tractcode_column
    • The name of the column in the dataframe containing census tractcode/blockcode data as a string.

Description

Takes a pandas dataframe and the name of the dataframe column containing census tractcode/blockcode data as arguments. Parses tractcode data to determine which state a tractcode is located in. Returns the same pandas dataframe from the argument with additional 'State' and 'State Initial'columns. Please note in version 0.0.2 the cendas.state_abb_column method has been discontinued and its functionality has been absorbed by the cendas.state_column method.

Why This Method is Useful

Some datasets provided by US government agencies only provide census tract codes with no reference to where exactly they are located.
