cendas.merge

cendas.merge(data_frame_list: list)

Parameters/Arguments

  • data_frame_list
    • A list of pandas dataframes.

Description

Takes a list of pandas dataframes as an argument. Performs a SQL-style inner join for all the dataframes in the list. Returns a single dataframe where all the dataframes in the list have been merged. This method can be used on a list of 2 or more pandas dataframes.

The argument assumes the following:

  • The user wants to perform a SQL-style inner join where the intersection of keys from the dataframes will be used.

Why This Method is Useful

Currently, the pandas.merge() method only allows the user to merge 2 dataframes at a time. However, this becomes highly inefficient when a user needs to merge many different dataframes. A good example of this is a user who is trying to merge dataframes containing data from different years into a single dataframe.
