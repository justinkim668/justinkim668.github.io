cendas.geoid_to_tract

cendas.geoid_to_tract(data_frame: pandas.DataFrame(), geoid_column: str)

Parameters/Arguments

  • data_frame
    • A pandas dataframe.
  • geoid_column
    • The name of the column in the dataframe containing census geoid data as a string.

Description

Takes a pandas dataframe and the name of the dataframe column containing census geoid data as arguments. Parses geographic identifier (geoid) codes and extracts census tractcodes. Returns the same dataframe with a "tractcode" column in the dataframe with the census tract codes.

Why This Method is Useful

Many datasets provided by U.S. government agencies have a "geoid" column where all values start with '1400000US' followed by a census tract code. This function effectively extracts the census tract codes from the geoid values.
