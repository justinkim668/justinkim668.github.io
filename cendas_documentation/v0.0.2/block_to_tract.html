cendas.block_to_tract

cendas.block_to_tract(data_frame: pandas.DataFrame(), block_code_column: str)

Parameters/Arguments

  • data_frame
    • A pandas dataframe.
  • block_code_column
    • The name of the column in the dataframe containing census blockcode data as a string.

Description

Takes a pandas dataframe and the name of the dataframe column containing blockcode data as arguments. Parses blockcode data to extract tractcodes from the blockcodes. Returns a dataframe with the an additional 'Tractcode' column.

The method assumes the following:

  • The user's blockcode data is type string.
  • The user's blockcode data is 15 digits long (Some datasets will have a column for block codes where the block codes are only 4 digits long -- this is not the case where the cendas_block_to_tract method would be used.).

Why This Method is Useful

Many research endeavors conduct analyses at the broader tract level in addition to the more specific block level (blocks are areas within tracts). However, many datasets only provide 15 digit blockcode data. The cendas.block_to_tract method allows a user to extract tractcode data from the blockcode data when the tractcode data is unavailable in a dataset.
